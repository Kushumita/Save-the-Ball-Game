# 🎮 Save The Ball Game (C++)

Save The Ball is a simple arcade-style C++ game where the player controls a paddle to keep the ball from falling. The objective is to keep the ball bouncing as long as possible and achieve the highest score.

---

## 🚀 Features

* Built using C++
* Real-time ball movement
* Paddle control using keyboard
* Collision detection
* Score tracking
* Game over condition
* Lightweight and fast

---

## 🕹️ How to Play

* Start the game
* Move the paddle left or right
* Catch the ball using the paddle
* Don't let the ball fall down
* Score increases as long as you survive
* Game ends when the ball falls

### 🎯 Controls

* Left Arrow  → Move Left
* Right Arrow → Move Right
* ESC → Exit Game

---

## 🛠️ Tech Stack

* C++
* Object-Oriented Programming
* Graphics Library (graphics.h)

---


## ▶️ How to Run

### 1. Clone Repository

```
git clone https://github.com/Kushumita/Save-the-Ball-Game.git
```

### 2. Navigate to Folder

```
cd Save-the-Ball-Game
```

### 3. Compile

```
g++ SaveTheBall.cpp -o game
```

### 4. Run

```
./game
```
